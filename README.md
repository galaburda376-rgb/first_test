# Smart Recorder (MVP start)

Стартовая версия проекта для разработки мобильного приложения «умный диктофон».

## Что уже сделано

- Создан базовый Flutter-каркас приложения.
- Есть экран списка записей.
- Есть экран создания записи (пока заглушка, без реальной записи аудио).
- Добавление демо-записи в список для проверки пользовательского потока.

## Ближайшие шаги

Подробный поэтапный план находится в `docs/DEVELOPMENT_PLAN.md`.
- Пошаговая инструкция для старта и запуска: `docs/STEP_BY_STEP_GUIDE_RU.md`.

## Запуск

```bash
cd mobile_flutter
flutter pub get
flutter run
```

## Первый рабочий сценарий

1. Открыть приложение.
2. Нажать `+`.
3. Нажать `Сохранить демо-запись`.
4. Вернуться на экран списка и увидеть новую запись.
