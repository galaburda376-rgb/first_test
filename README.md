# Smart Recorder (MVP start)

Стартовая версия проекта для разработки мобильного приложения «умный диктофон».

## Что уже сделано

- Создан базовый Flutter-каркас приложения.
- Есть экран списка записей.
- Есть экран создания записи с таймером и сохранением заметки (пока без записи микрофона в файл).
- Сохраненная заметка попадает в список с длительностью и статусом обработки.

## Ближайшие шаги

Подробный поэтапный план находится в `docs/DEVELOPMENT_PLAN.md`.
- Пошаговая инструкция для старта и запуска: `docs/STEP_BY_STEP_GUIDE_RU.md`.
- Чеклист по синхронизации ветки и обновлению PR: `docs/PR_WORKFLOW.md`.

## Запуск

```bash
cd mobile_flutter
flutter pub get
flutter run
```

## Первый рабочий сценарий

1. Открыть приложение.
2. Нажать `+`.
3. Нажать `Старт`, затем `Остановить`.
4. Нажать `Сохранить запись`.
5. Вернуться на экран списка и увидеть новую запись с длительностью и статусом.
